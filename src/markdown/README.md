## Headers

Levels 1-6 can be implemented with hashtags

| Markdown | Preview |
| --- | --- |
| `# Hello` | <h1>Hello</h1> |

Placing a horizontal line below a new line item, at least 3 wide, turns it into a header 1 with a horizontal line.

```
Hello
==========
```
